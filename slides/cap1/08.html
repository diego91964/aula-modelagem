Motivação

Pense na seguinte situação: Temos um sistema bancário que possui uma interface utilizada pelo gerente do banco, uma interface utilizada em um caixa eletrônico e uma interface mobile. Todas estas interfaces precisam se comunicar com um banco de dados hipotético (bank-db) que utiliza uma tecnologia muito antiga não suportada. Você precisa apresentar uma solução para este cenário para o dono do banco, um renomado artista sem qualquer experiência em desenvolvimento de softwares.
